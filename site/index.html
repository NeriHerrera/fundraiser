Una página de muestra, lista para Traefik

Este sitio estático está dockerizado y preparado para ser publicado en fundraiser.com.ar detrás de un proxy Traefik.

Ver características Cómo funciona

Características principales

Dockerizada

Construida con Nginx en una imagen ligera. Despliegue simple y reproducible.

Traefik-ready

Incluye labels para enrutamiento HTTP/HTTPS, redirección y TLS mediante un certresolver.

Estática y veloz

HTML, CSS y un poco de JavaScript. Ideal para landings y documentación.

Fácil de personalizar

Estructura simple en la carpeta /site. Edita estilos, textos e imágenes a gusto.

Cómo funciona

  1. Construimos la imagen: docker compose build.
  2. Publicamos el servicio con labels de Traefik: docker compose up -d.
  3. Traefik enruta el dominio fundraiser.com.ar hacia este contenedor.

¿Listo para personalizar?

Modifica el contenido en /site y despliega de nuevo. ¡Eso es todo!
